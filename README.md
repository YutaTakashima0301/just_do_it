# 進捗がないときにDo It おじさんが励ましてくれる

## 仕組み

一定期間PCに入力がないと判断した場合，Do It おじさんのyoutube動画を開きます．今回はpythonでシステムを構築しました．

## 動かし方

ターミナルでpynputをインストール
```bash
pip install pynput
```
起動
```bash
python main.py
```

## 各種設定
main.pyの13行目で入力を何秒間待つかを設定できます
